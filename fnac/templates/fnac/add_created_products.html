Add Created Products

Upload a Mirakl product export to mark products as created. These products will then be included in updates.

To produce a product export login to mirakl.net and navigate to My Inventory > Catalogue Management. Go to the Published tab and select all the products. Ensure you select all products, not just the ones on the current page. Click Generate Export File. Once the file has been created click Download Last Export File Generated and upload the resulting file below.
